## Project Installation Guide

Clone the project

```bash
  git clone https://github.com/sandeshjadhav5/Bath-Body-Works-E-commerce-Website-Clone.git
```

Go to the project directory

```bash
  cd bath-and-body-works
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
